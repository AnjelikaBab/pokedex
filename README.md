# pokedex
pokemon wiki using Node.js, GraphQL, ReactJS

In order to run this application: 

Clone repo

Starting Server:
Cd into server folder and run:
<br /> npm run start

Running Local Host:
Cd into client folder and run: <br /> 
npm install <br /> 
npm start
